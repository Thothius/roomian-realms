Race Banner Generator

This page generates banner images for each race in the RG2 mod. Follow these steps:

  1. Right-click on each banner and select "Save image as..."
  2. Save each image with the corresponding race name (e.g., "wormlings.png", "bloodelves.png", etc.)
  3. Place all images in the "images" folder of your mod

The banners are styled to match each race's theme and religious affiliation.
